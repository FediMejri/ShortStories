# ShortStories (Description)
This app is a social network for creating public and private short life stories.  
Users can login with their google accounts and create stories then set them to public or private and allow comments on them or not.  
Users maintain their stories through a dashboard.  

# Technical Description  
*) Creation of the server using Node.js and Express.  
*) Connection and Management of the database using MongoDB.  
*) Working with remote MongoDB deployment using MongoDB ATLAS.  
*) Implement authentication including local & Google OAuth strategies.  
*) Handling views and layouts on the server using "Handlebars".  
*) Used modules : express, mongoose, handlebars, passport, passport-google-oauth20, materializecss.  
*) Prepare and deploy the app to production with Heroku.
